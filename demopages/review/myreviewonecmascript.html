In this video, I learn about the history of JavaScript. JavaScript's history has been a rollercoaster ride with many ups and downs. Creating a programming language where developers can easily read and understand was merely impossible not until Brendan Eich invented it in just 10 days and named it as Mocha which is renamed to LiveScript then JavaScript. JavaScript refused to be added to the Microsoft web browser which is the Internet Explorer, as both company has their own web browser and programming language. In this case, most of the developers are having a hard time in supporting both web browsers. To put a stop to the confusion, Netscape submitted JavaScript to ECMA International for a standard specification. The International Organization for Standardization (ISO) published ECMAScript, which became the standard specification. Up to 2017, new versions were released. JavaScript had become the language of the web by the time ECMAScript was released, and it was the most important language to learn because it could be used in both the front-end and the back-end.
